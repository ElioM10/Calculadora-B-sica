# Descripción del Proyecto

Calculadora básica desarrollada en Java Swing inspirada en las funcionalidades de la calculadora típica de windows, diseñada para realizar operaciones aritméticas simples a través de una interfaz gráfica intuitiva.
El proyecto tiene como objetivo aplicar conceptos fundamentales de Programación Orientada a Objetos, manejo de eventos, y diseño de interfaces gráficas en Java.

La aplicación permite al usuario realizar cálculos de manera sencilla, validando entradas y mostrando los resultados en tiempo real.

## Tecnologías utilizadas

Java

Java Swing

IntelliJ IDEA

Git y GitHub

## Funcionalidades

Suma, resta, multiplicación y división

Interfaz gráfica amigable

Manejo de eventos con ActionListener

Validación de entradas numéricas

Limpieza y reinicio de operaciones

## Objetivo del proyecto

Reforzar los conocimientos en Java SE, especialmente en el desarrollo de aplicaciones de escritorio, aplicando buenas prácticas de código y una estructura clara.
